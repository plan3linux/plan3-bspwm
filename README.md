# BSPWM Config for Plan3
![Screenshot of BSPWM Config](plan3-bspwm.png)

This is the BSPWM configuration that will eventually be added in the forthcoming Arch-Based Linux Distro, Plan3.

## Prerequisites
- alacritty
- bspwm
- sxhkd
- picom
- conky
- rofi
- network-manager-applet
- numlockx
- polkit-gnome
- xfce4-notifyd
- nitrogen
- volumeicon
- JetBrains Mono Nerd
